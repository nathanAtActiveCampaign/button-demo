# Accordion

Accordions allow the user to expand and collapse content. They put the user in control of what to read and what to ignore.

## Variations

### Default accordion

Use the default accordion wherever possible to expand and collapse content. 

```tsx copy
import Accordion from '@activecampaign/camp-components-accordion';

<Accordion
  title="Accordion Title Text"
>
  <Accordion.Item
    title="Example Title 1, Plain text"
    panel="Hi i'm a panel here. Pass anything you want in here!"
  />
  <Accordion.Item
    flush
    title={() => <div>Example Title 2, Flush</div>}
    panel={() => (
      <div>a panel here, could be a component, notice there is no panel padding</div>
    )}
  />
  <Accordion.Item
    title={() => <div>Example Title 3, Component</div>}
    panel={'a panel here, could be a component'}
  />
  <Accordion.Item
    title={() => 'Example Title 4, Do not persistent input data'}
    panel={getPanelWithInput}
  />
  <Accordion.Item
    persistPanelContent
    title={() => 'Example Title 5, Persistent input data'}
    panel={getPanelWithInput}
  />
</Accordion>

function getPanelWithInput() {
  return (
    <label>
      Edit this field <input type="text" />
      <br />
      Then close and re-open this panel to see your value
    </label>
  );
}
```

### Side panel accordion

### Accessibility

#### Keyboard interactions

* `tab` to switch between accordion items  
* When the accordion item is in `focus`, `space` or `enter` to expand or collapse the accordion item
* Interactive elements (such as a link or button) within an expanded accordion item integrate into the tab order by default
